Titolo e sottotitolo hanno lo stesso stile (guarda il sorgente HTML utilizzando gli strumenti di sviluppatore del browser). Avendo deciso di aggiungere lo stile attraverso l'attributo style siamo costretti a ripetere lo stile ogni volta.

Inoltre, sempre guardando il codice sorgente, noterai che ogni volta che è necessario assegnare uno stile il codice HTML diventa difficile da leggere perché interrotto dai vari stili.
